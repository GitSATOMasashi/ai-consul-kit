# ai-consul-kit

AIと会話するだけで、プロジェクトの立ち上げを進められるツールです。

## はじめに（初回セットアップ）

### 1. Cursor をインストール

1. https://cursor.com にアクセス
2. 「Download」をクリック
3. ダウンロードしたファイルを開き、アプリケーションフォルダにドラッグ
4. Cursor を起動し、アカウントを作成（メールアドレスで登録）

### 2. このリポジトリをダウンロード

1. Cursor を開く
2. 上部メニューから「ターミナル → 新しいターミナル」を選択
3. 表示されたターミナル（黒い画面）に、以下をそのままコピー＆ペースト:

```
cd ~/Desktop && git clone https://github.com/GitSATOMasashi/ai-consul-kit.git
```

### 3. Cursor でフォルダを開く

1. Cursor のメニューから「ファイル → フォルダーを開く」
2. デスクトップにある `ai-consul-kit` フォルダを選択

### 4. AI に話しかける

右側のチャット欄に「こんにちは」と送ってみてください。AIが案内してくれます。

---

## できること（スキル一覧）

| スキル | やること | こう話しかけてください |
|---|---|---|
| タスクリスト生成 | やりたいことを整理してタスクリストを作る | 「始めたい」「タスクリスト作って」 |
| 要件定義 | 何を作るか、機能や仕様を整理する | 「要件定義したい」「機能を決めたい」 |
| セットアップガイド | Cursor の日本語化や基本操作を案内する | 「使い方がわからない」「セットアップしたい」 |

---

## 基本の使い方

- **「次何する？」** と聞けば、AIが次のタスクに誘導してくれます
- **困ったら「使い方がわからない」** と言えば、操作を案内してくれます
- 何を話しかけていいかわからないときは、**「始めたい」** と送ればOKです

---

## 最新版にするには

ターミナルで以下を実行してください:

```
git pull
```

わからなければ、チャットで「更新したい」と話しかけてください。
