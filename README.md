# ai-consul-kit

AIと会話するだけで、プロジェクトの立ち上げを進められるツールです。

## はじめに（初回セットアップ）

### ステップ 1: Cursor をインストール

1. https://cursor.com にアクセス
2. 「Download」をクリック
3. ダウンロードしたファイルを開き、アプリケーションフォルダにドラッグ
4. Cursor を起動し、アカウントを作成（メールアドレスで登録）

### ステップ 2: Cursor でターミナルを開く

1. Cursor を起動する
2. キーボードで `Ctrl + `` ` を押す（ターミナルという黒い画面が下に出ます）

> ターミナルが開かない場合は、上のメニューから「ターミナル → 新しいターミナル」を選んでください。

### ステップ 3: 開発ツールをインストール

ターミナルに以下をコピー＆ペーストして Enter を押してください:

```
xcode-select --install
```

- ポップアップが表示されたら **「インストール」をクリック** してください
- インストールには **5〜10分** かかります。そのままお待ちください
- 完了したら、次のステップ 4 に進んでください

> 「既にインストールされています」と表示された場合は、そのままステップ 4 に進んでください。

### ステップ 4: Homebrew をインストール

ターミナルに以下をコピー＆ペーストして Enter を押してください:

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- **パスワード**を求められたら、Mac のログインパスワードを入力してください（文字は表示されませんが入力されています）
- 「**Press RETURN/ENTER to continue**」と表示されたら Enter を押してください
- インストールには **数分** かかります。そのままお待ちください

Apple Silicon Mac（M1/M2/M3/M4）の場合、完了後に以下も実行してください:

```
eval "$(/opt/homebrew/bin/brew shellenv)" && echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
```

> 「既にインストールされています」と表示された場合は、そのままステップ 5 に進んでください。

### ステップ 5: GitHub CLI をインストール

ターミナルに以下をコピー＆ペーストして Enter を押してください:

```
brew install gh
```

自動で終わるのでそのままお待ちください。

### ステップ 6: GitHub にログイン

ターミナルに以下をコピー＆ペーストして Enter を押してください:

```
gh auth login --web --git-protocol https
```

- **ブラウザが自動で開きます**。GitHub にログインしてください
- ターミナルに **8桁のコード** が表示された場合は、ブラウザの画面にそのコードを入力してください
- ブラウザで **「Authorize」ボタン** を押してください
- 認証が終わると自動でターミナルに戻ります

認証が完了したら、以下も実行してください:

```
gh auth setup-git
```

### ステップ 7: リポジトリをダウンロード

ターミナルに以下をコピー＆ペーストして Enter を押してください:

```
git clone https://github.com/GitSATOMasashi/ai-consul-kit.git
```

数秒で完了します。完了したら:

1. Cursor のメニューから **「ファイル → フォルダーを開く」**
2. `ai-consul-kit` フォルダを選択
3. 右側のチャット欄に **「こんにちは」** と送ってみてください。AIが案内してくれます！

## できること（スキル一覧）

| スキル | やること | こう話しかけてください |
|---|---|---|
| タスクリスト生成 | やりたいことを整理してタスクリストを作る | 「始めたい」「タスクリスト作って」 |
| ユーザー体験設計 | 使い方の流れや画面遷移を整理する | 「ユーザー体験を決めたい」「流れを考えたい」 |
| 要件定義 | 何を作るか、機能や仕様を整理する | 「要件定義したい」「機能を決めたい」 |
| FAQ・ナレッジ整理 | よくある質問と回答を体系的にまとめる | 「FAQを作りたい」「質問を整理したい」 |
| セットアップガイド | Cursor の日本語化や基本操作を案内する | 「使い方がわからない」「セットアップしたい」 |

---

## 基本の使い方

- **「次何する？」** と聞けば、AIが次のタスクに誘導してくれます
- **困ったら「使い方がわからない」** と言えば、操作を案内してくれます
- 何を話しかけていいかわからないときは、**「始めたい」** と送ればOKです

## 最新版にするには

ターミナルで以下を実行してください:

```
git pull
```

わからなければ、チャットで「更新したい」と話しかけてください。

## よくあるトラブル

### 「パスワードを入力してください」と出た

Mac のログインパスワード（PC起動時に入力するパスワード）を入力してください。セキュリティのため、入力しても画面に文字は表示されませんが、入力されています。そのまま Enter を押してください。

### パスワードがわからない

1. 画面左上の Apple メニュー → 「システム設定」→「ユーザとグループ」→「パスワードを変更」で変更できます
2. それでもわからない場合は、Mac のログイン画面でわざと間違ったパスワードを3回入力すると、Apple ID を使ってリセットできます
