# 社内ツール テンプレート

業務効率化・社内システム開発プロジェクトのタスクテンプレート。

## タスク候補

- [ ] 要件定義: 現在の業務フローと課題を整理し、ツールに求める機能を決める
- [ ] 業務フローの整理: 現状の作業手順を書き出し、自動化・効率化できるポイントを見つける
- [ ] 機能設計: ツールがやることを具体的に決める
- [ ] 技術選定: 使うツールやサービスを決める（スプレッドシート活用、専用アプリなど）
- [ ] プロトタイプ作成: まず最小限の機能で動くものを作る
- [ ] 社内テスト: 実際の業務で使ってみて、問題がないか確認する
- [ ] 改善: テストで見つかった問題を修正する
- [ ] 展開: チーム全体に導入する

## 適用基準

以下のようなプロジェクトに使う:
- 社内の業務を効率化するツール
- データ管理、レポート自動生成
- ワークフローの自動化
- Google スプレッドシートや GAS を活用した仕組み
