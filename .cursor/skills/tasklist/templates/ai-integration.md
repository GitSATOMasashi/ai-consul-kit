# AI活用・導入 テンプレート

AI機能の導入・活用プロジェクトのタスクテンプレート。

## タスク候補

- [ ] 要件定義: AIで解決したい課題と、期待する効果を明確にする
- [ ] 活用シーンの洗い出し: どの業務でAIを使うかリストアップする
- [ ] AI手法の選定: どのAIサービス・モデルを使うか決める（ChatGPT、Claude、画像生成AIなど）
- [ ] データの準備: AIに必要なデータ（プロンプト、学習データ、参考資料など）を整理する
- [ ] プロンプト設計: AIへの指示文を作成・テストする
- [ ] プロトタイプ作成: まず小さく試して効果を確認する
- [ ] 評価・改善: 結果を確認し、プロンプトや設定を調整する
- [ ] 業務への組み込み: 日常業務のフローに組み込む

## 適用基準

以下のようなプロジェクトに使う:
- ChatGPT や Claude を業務に導入したい
- AIを使った自動化（文章生成、分析、翻訳など）
- AIチャットボットの構築
- 既存システムへのAI機能追加
