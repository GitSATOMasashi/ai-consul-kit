# Webサービス開発 テンプレート

標準的なWebサービス/Webアプリ開発プロジェクトのタスクテンプレート。

## タスク候補

- [ ] 要件定義: サービスの目的・機能・ターゲットユーザーを明確にする
- [ ] ユーザー体験の設計: ユーザーがサービスをどう使うかの流れを考える
- [ ] 画面デザイン: 主要な画面のイメージを作る
- [ ] 技術選定: 使う技術やサービスを決める
- [ ] データ設計: 保存するデータの種類と構造を決める
- [ ] 開発環境のセットアップ: 開発を始められる状態にする
- [ ] 主要機能の実装: メインの機能を作る
- [ ] テスト: 作ったものが正しく動くか確認する
- [ ] デプロイ: サービスを公開する
- [ ] 運用開始: 実際に使い始める

## 適用基準

以下のようなプロジェクトに使う:
- ユーザー向けのWebアプリケーション
- ECサイト、予約サイト、マッチングサービスなど
- ランディングページやコーポレートサイト
