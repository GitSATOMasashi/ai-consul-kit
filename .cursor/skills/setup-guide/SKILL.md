# セットアップガイド

## トリガー

「セットアップしたい」「使い方がわからない」「操作方法を教えて」「初期設定」「gitが使えない」「認証エラー」等

## 前提

- クライアントは Cursor でこのリポジトリを開いた状態
- README の setup.sh による初回セットアップは完了済みの想定
- このスキルは対話形式で進める。一度に全部説明せず、ステップごとに確認を挟む

## 手順

### Step 0: 環境のトラブル対応（必要な場合のみ）

クライアントが「git が使えない」「認証エラーが出た」等の問題を報告した場合、以下を順に確認する。

**git が使えない場合:**
ターミナルで以下を実行してもらう:
```
git --version
```
エラーが出たら「ポップアップが出たら『インストール』を押してください」と案内する。

**GitHub 認証エラーの場合:**
ターミナルで以下を実行してもらう:
```
gh auth status
```
認証されていない場合:
```
gh auth login --web --git-protocol https
gh auth setup-git
```
「ブラウザが開くのでGitHubにログインしてください」と案内する。

**gh コマンドが見つからない場合:**
setup.sh を再実行してもらう:
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/GitSATOMasashi/ai-consul-kit/main/setup.sh)"
```

問題が解決したら Step 1 に進む。問題がなければ Step 0 はスキップ。

### Step 1: 日本語化と推奨設定

**日本語化:**

1. 「`Cmd + Shift + P` を押してください（コマンドパレットが開きます）」
2. 「`Configure Display Language` と入力して、表示された候補をクリックしてください」
3. 「`ja` を選択してください。Cursor が再起動して日本語になります」

**推奨設定:**

1. 「左下の歯車アイコン → 設定 を開いてください」
2. 「Auto Save を `afterDelay` に変更しましょう（ファイルが自動保存されるようになります）」

設定が終わったら「ここまで大丈夫ですか？」と確認する。

### Step 2: Cursor の基本操作ガイド

以下を実際に操作しながら案内する:

1. **チャットの使い方**:
   - 「右サイドバーにチャット欄があります。ここに話しかけるとAIが答えてくれます」
   - 「試しに『こんにちは』と送ってみてください」

2. **Agentモードの確認**:
   - 「チャット欄の上部に『Agent』と表示されていれば OK です」
   - 「Agent モードでは、AIがファイルを作ったり編集したりできます」

3. **ファイル参照の使い方**:
   - 「チャットで `@` をつけるとファイルを指定できます。後で使うかもしれないので覚えておいてください」

### Step 3: 完了確認

以下のチェックリストを使って、セットアップが完了しているか確認する:

- [ ] Cursor が日本語表示になっている
- [ ] Auto Save が有効になっている
- [ ] チャットで AI と会話できる
- [ ] Agent モードが選択されている

すべて OK なら:
「セットアップ完了です！次のステップに進みましょう。」

## 注意事項

- 既に日本語化済み・設定済みの場合はスキップして Step 2 に進む
- クライアントが操作に迷ったら「画面のスクリーンショットを送ってください」と案内する
- 環境トラブルで詰まった場合は、setup.sh の再実行を最初に試す
